R package for modeling animal movement with continuous-time correlated random walk models

Welcome to crawl.

crawl is an R package for the analysis of animal movement from telemetry data using continuous-time correlated random walks (i.e, (c)orrelated (ra)ndom (w)alk (l)ibrary of R functions). It's not really a library, it's a package, but crawp didn't sound as good.

The release version is on CRAN. The development version is located on github and can be installed with the devtools package via

library(devtools)
install_github("NMML/crawl")
